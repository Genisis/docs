The span_multi query allows you to wrap a multi term query (one of fuzzy, prefix, term range or numeric range query) as a span query, so it can be nested. Example:

{
    "span_multi":{
        "match":{
            "prefix" : { "user" :  { "value" : "ki" } }
        }
    }
}

A boost can also be associated with the query:

{
    "span_multi":{
        "match":{
            "prefix" : { "user" :  { "value" : "ki", "boost" : 1.08 } }
        }
    }
}

The span_multi query is supported from version 0.90.1
